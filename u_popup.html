0. 공통

* Popup

· 상황에 맞게 문구를 변경하여 사용할 수 있음

0-1. 질문 (ex. ○○ 하시겠습니까?)
0-2. 성공 (ex. 저장 완료되었습니다. / 적용 완료되었습니다.)
0-3. 실패 (ex. 예약 생성 중 오류가 발생했습니다. / 공지 사항 생성 중 오류가 발생했습니다.)
0-4. 탈출 (ex. 작성 중인 내용이 있습니다. 저장하지 않고 나가시겠습니까?)
0-5. 기타 알림 (ex. ○○을 입력해 주세요.)

1. 예약 생성

* Modal

· 월간 및 주간 달력에서 예약 가능 객실 클릭 시 / 예약 생성 링크 클릭 시
· 파트너의 경우, 채널은 직판으로 고정이고 관리자 메모가 없으며 1-3을 사용할 수 없다.

1-1. 달력에서 예약 가능 객실을 선택한 경우 : 입실일은 달력에서 클릭한 날짜, 퇴실일은 입실일 다음날, 객실 타입 및 번호는 클릭한 객실을 기본으로 입력
1-2. 1-1과 동일하되, 객실이 추가 되었을 때의 모습
1-3. 예약 생성 링크 클릭 시 (관리자 전용) : 달력에 진입하지 않아도 링크를 통해서 접근할 수 있는 팝업. 입실일은 예약 일시, 퇴실일은 입실일 다음날을 기본으로 하고, 숙소를 선택할 수 있다.

* Popup

· '예약 생성'에서 '저장' 버튼 클릭 시
· 알림톡으로 가는 메시지 내용이 표시되고, 알림톡 전송 여부를 선택할 수 있다. (기본 값으로 체크가 되어 있다.)
· 파트너의 경우, 전송 여부를 선택하는 체크 박스가 없다. (항상 알림톡 발송)
· '돌아가기' 클릭 시, '예약 생성' 화면으로 복귀 (작성 했던 내용 유지)

1-4. 예약 생성 메시지 Popup 예시

2. 예약 조회

* Modal

· 월간 및 주간 달력, 예약 목록 페이지에서 '예약 완료', '입금 대기'에 해당하는 예약 건을 클릭 시
· 파트너의 경우, 관리자 메모 및 작성, 수정 로그를 볼 수 없다.
· 파트너의 경우, '예약 취소', 'SMS 발송 및 발송 여부 확인' 기능이 없다.

2-1. 단일 객실 / 일정 나누기 없는 경우
2-2. 복수 객실 / 일정 나누기 있는 경우
2-3. 파트너가 예약 건을 조회 하는 경우

3. 예약 수정

* Modal

· '예약 조회'에서 '예약 수정' 버튼 클릭 시
· 관리자는 모든 예약 건에 대해서 모든 항목을 수정할 수 있다.
· 파트너는 관리자가 생성한 예약 건에 대해서 일부만 수정할 수 있다.
· 파트너는 본인이 작성한 예약 건에 대해서 관리자 메모를 제외한 모든 항목을 수정할 수 있다. (관리자 메모 부분이 뜨지 않음)

3-1. 단일 객실 / 일정 나누기 없는 경우
3-2. 복수 객실 / 일정 나누기 있는 경우
3-3. 관리자가 생성한 예약 건을 파트너가 수정 하는 경우

* Popup

· '예약 수정'에서 '저장' 버튼 클릭 시
· 알림톡으로 가는 메시지 내용이 표시되고, 알림톡 전송 여부를 선택할 수 있다. (기본 값으로 체크가 되어 있다.)
· 파트너의 경우, 전송 여부를 선택하는 체크 박스가 없다. (항상 알림톡 발송)

3-4. 예약 수정 메시지 Popup 예시

4. 예약 취소 생성

* Modal

· '예약 조회'에서 '예약 취소' 버튼 클릭 시
· '예약 생성' 및 '예약 수정'에서 작성한 '쉼플 메모'의 내용을 그대로 가지고 오고, 수정할 수 있다.
· 파트너의 경우, '예약 취소'를 발생시킬 수 없으므로, 해당 Modal을 볼 수 없다.
· (일반적으로 예약 취소는 입실 전에 이뤄지는 경우가 대부분이므로, 일정 나누기가 발생된 예약 건이 취소되는 경우는 드물다.)

4-1. 예약 취소 생성 Modal 예시

* Popup

· 알림톡으로 가는 메시지 내용이 표시되고, 알림톡 전송 여부를 선택할 수 있다. (기본 값으로 체크가 되어 있다.)

8. 객실 이동

* Popup

· 주간 달력에서 '객실 이동'을 통해 '적용' 클릭 시
· 알림톡을 발송하지 않는다.
· 객실 이동은 같은 타입, 같은 입실일이면서, 다른 예약 건들과 겹치지 않는 경우에만 가능하다.
· 주간 달력에서 가능하고, 한 번에 하나의 타입만 객실 이동(호수 변경)을 할 수 있다. (객실 타입명 아래 '객실 이동' 버튼 클릭을 통해 가능)

9. SMS 발송

* Popup

· 고객(예약자)에게 보내는 메시지

9-1. 반자동 Popup (예약 조회, 예약 취소 조회, 관리자 페이지의 예약 목록에서 'SMS' 버튼을 클릭하면 팝업이 뜨고 여기에서 발송)
